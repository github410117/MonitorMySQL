# MonitorMySQL  
MonitorMySQL Log监听MySQL的日志Log  
用于监听本地`MySQL`的操作日志  
### Using (使用方法)
`python3 MonitorMySQL.py`  
如果提示库不存在请安装对应的库  
`pip install pymysql`或者`conda install pymysql`
